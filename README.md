# HR-Employee-Attrition-Rate-Power-BI-Dashboard
This project is an interactive Power BI dashboard designed to analyze employee data and understand attrition patterns within an organization.
The dashboard provides insights into workforce composition, employee performance, and attrition trends. It helps HR teams identify key factors influencing employee turnover and supports data-driven decision-making.
Insights Delivered
1.Identifies high attrition risk groups
2.Analyzes workforce demographics
3.Highlights salary distribution trends
4.Supports HR strategy and retention planning
